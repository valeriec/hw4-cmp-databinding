Bind it!

  1. Crea tres nuevos componentes: GameControl, Odd y Even
  2. El componente GameControl debe tener botones para iniciar y detener el juego.
  3. Al iniciar el juego, debe emitirse un evento (con un número creciente) cada segundo (ref = setInterval())
  4. El evento debe ser escuchable desde fuera del componente.
  5. Al detener el juego, no se deben emitir más eventos (clearInterval(ref))
  6. Se debe crear un nuevo componente impar para cada número impar emitido, lo mismo debe suceder para el componente par (en números pares)
  7. Simplemente emita Impar - NÚMERO o Par - NÚMERO en los dos componentes
  8. Diseñe el elemento (por ejemplo, párrafo) que contiene su texto de salida de manera diferente en ambos componentes
